# 📋 وثيقة تسليم المشروع
## نظام الانتخابات الإلكترونية - اتحاد طلاب كلية التربية النوعية

---

## 🎯 نظرة عامة على المشروع

نظام انتخابات إلكترونية متكامل وآمن لإدارة العملية الانتخابية لاتحاد الطلاب بشكل كامل، من مرحلة الترشح وحتى إعلان النتائج النهائية.

---

## ✨ المميزات الرئيسية

### 🔐 نظام مصادقة آمن
- تسجيل دخول بكلمة المرور
- تسجيل دخول بكود OTP (يُرسل للبريد الإلكتروني)
- حماية كاملة للبيانات

### 👨‍🎓 واجهة الطالب
- **لوحة تحكم شخصية** لمتابعة حالة الترشح والتصويت
- **تقديم الترشح** لـ 9 لجان مختلفة
- **التصويت الإلكتروني** بشكل آمن ومشفر
- **تقديم الطعون** على المرشحين

### 👔 لوحة الإدارة
- **إحصائيات شاملة** (عدد الطلاب، المرشحين، الأصوات، الطعون)
- **متابعة نسبة الحضور** (النصاب) بشكل مباشر
- **إدارة المراحل الانتخابية** (11 مرحلة)
- **مراجعة طلبات الترشح** (قبول/رفض/استبعاد)
- **فحص الطعون** والرد عليها
- **إدارة صلاحيات المستخدمين**
- **بدء انتخابات جديدة** (تصفير النظام)

## 🚀 كيفية تشغيل النظام

### المتطلبات
- Node.js (الإصدار 18 أو أحدث)
- حساب Supabase (قاعدة البيانات)
- حساب Vercel (للنشر) - اختياري

### خطوات التشغيل المحلي

```bash
# 1. الدخول لمجلد المشروع
cd voting-system

# 2. تثبيت المكتبات
npm install

# 3. تشغيل النظام
npm run dev
```

ثم افتح المتصفح على: **http://localhost:3000**

## 🔧 إعداد قاعدة البيانات

### الخطوة 1: إنشاء مشروع Supabase
1. اذهب إلى [supabase.com](https://supabase.com)
2. أنشئ مشروع جديد
3. احفظ الـ URL والـ Anon Key

### الخطوة 2: تشغيل ملف الإعداد
1. افتح SQL Editor في Supabase
2. انسخ محتوى الملف: `src/lib/supabase/clean_setup_and_seed.sql`
3. شغّل الكود

### الخطوة 3: تحديث ملف البيئة
عدّل ملف `.env.local`:
```
NEXT_PUBLIC_SUPABASE_URL=رابط_مشروعك
NEXT_PUBLIC_SUPABASE_ANON_KEY=المفتاح_الخاص_بك
```
